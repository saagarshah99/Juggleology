What are the different types of throws?

Each trick is denoted using a sequence of numbers, whereby each digit represents a different throw in the pattern. The table below outlines what each digit (siteswap number) represents...

Patterns are mathematically continuous and infinite.

Odd number = opposite hand (cascade).

Even number = same hand (fountain).

How do we form tricks from these different types of throws?

You cannot just randomly string numbers together - there is some logic behind this which is necessary to get correct in order to allow another juggler to decipher the trick and try it out themselves.
